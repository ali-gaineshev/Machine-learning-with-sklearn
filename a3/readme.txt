These files are for assignment 3 seng474. 

Main files:

k_mean.py - K means algorithm with random and k-means++ initialization methods
To run - python k_mean.py

agglomerative_clustering.py - hierarchical agglomerative clustering with single and average linkages
To run - python agglomerative_clustering.py

Additional files:

data.py - processes the datasets given that "dataset1.csv" and "dataset2.csv" are present in data folder

clusters.py - class for clusters. Used in k_means.py

meshlab_visual.py - visualization of data with 3d points. Generates .xyz file.
